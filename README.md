# 🌿 BaqueMix - Roda de Samambaia

Une application web interactive conçue pour arranger, visualiser et diriger les morceaux de Maracatu. Pensée pour les besoins du maracatu **Samambaia**, elle permet d'éditer les lignes de percussions traditionnelles et de gérer facilement la répartition des chants.

🌍 **Accéder à l'application :** [Lancer l'interface](https://julianbzh-coder.github.io/roda-samambaia/)

## ✨ Fonctionnalités principales

- **Misturador (Table de mixage) 🎛️** : Ajustez le tempo (BPM), la métrique (Compassos en 2/4, 3/4, 4/4, 6/8, 12/8) et le volume global de l'arrangement.
- **Éditeur d'instruments 🥁** : Programmez visuellement les frappes pour chaque instrument du baque (Alfaia, Caixa, Gonguê, Agbê, Mineiro).
- **Gestionnaire de Voix 🎤** : Assignez et distinguez visuellement les parties vocales (Puxador en bleu foncé / Coro en bleu clair).
- **Extraction des paroles 📝** : Générez et exportez automatiquement les paroles (Letras) à partir de la timeline.
- **Sauvegarde et Navigation 💾** : Enregistrez vos arrangements et naviguez entre vos différents morceaux.

## 🚀 Comment l'utiliser ?

L'éditeur utilise un système de frappe au clavier intuitif pour programmer les rythmes :

**Règle d'or de la dynamique :**
- **Majuscule** = Coup fort / accentué
- **Minuscule** = Coup faible / note fantôme

**Raccourcis par instrument :**
- **Alfaia & Caixa :** `D`/`d` (Main droite), `G`/`g` (Main gauche)
- **Gonguê :** `G`/`g` (Grave), `A`/`a` (Aigu)
- **Agbê :** `G`/`g` (Gauche), `D`/`d` (Droite)
- **Mineiro :** `P`/`p` (Haut), `T`/`t` (Bas)

**Navigation :**
- Appuyez sur **Espace** pour avancer dans la mesure ou laisser un silence.
- Utilisez les **Flèches directionnelles (←/→)** pour vous déplacer dans la grille rythmique.

## 🛠️ Stack Technique

# 🌿 BaqueMix - Roda de Samambaia

Uma aplicação web interativa projetada para arranjar, visualizar e guiar as músicas de Maracatu. Desenvolvida sob medida para as necessidades da **Roda de Samambaia**, ela permite editar as linhas de percussão tradicionais e gerenciar facilmente a distribuição do canto.

🌍 **Acesse a aplicação:** [Abrir a interface](https://julianbzh-coder.github.io/roda-samambaia/)

## ✨ Principais Funcionalidades

- **Misturador 🎛️**: Ajuste o andamento (BPM), a fórmula de compasso (2/4, 3/4, 4/4, 6/8, 12/8) e o volume global do arranjo.
- **Editor de Instrumentos 🥁**: Programe visualmente os golpes para cada instrumento do baque (Alfaia, Caixa, Gonguê, Agbê, Mineiro).
- **Gerenciador de Vozes 🎤**: Atribua e diferencie visualmente as partes vocais (Puxador em azul escuro / Coro em azul claro).
- **Extração de Letras 📝**: Gere e exporte automaticamente as letras das músicas diretamente a partir do sequenciador.
- **Salvamento e Navegação 💾**: Salve seus arranjos localmente e navegue facilmente entre suas diferentes composições.

## 🚀 Como usar?

O editor utiliza um sistema de digitação intuitivo pelo teclado para programar os ritmos na grade:

**Regra de ouro da dinâmica:**
- **Maiúscula** = Golpe forte / acentuado
- **Minúscula** = Golpe fraco / nota fantasma

**Atalhos por instrumento:**
- **Alfaia & Caixa:** `D`/`d` (Mão direita), `G`/`g` (Mão esquerda)
- **Gonguê:** `G`/`g` (Grave), `A`/`a` (Agudo)
- **Agbê:** `G`/`g` (Esquerda), `D`/`d` (Direita)
- **Mineiro:** `P`/`p` (Cima), `T`/`t` (Baixo)

**Navegação:**
- Pressione a barra de **Espaço** para avançar na medida ou deixar um silêncio.
- Use as **Setas direcionais (←/→)** para se mover de um tempo a outro na grade rítmica.
